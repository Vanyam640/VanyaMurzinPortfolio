Statistical Analysis of Heart Disease Risk Factors

I developed a predictive analytics project using R to study heart disease risk factors across multiple chest pain categories. Engineered features from patient health data, performed correlation analysis and multiple linear regression, and evaluated model performance using correlation coefficients and R² values. Sleep Stage Characterization

I built an end-to-end sleep stage classification pipeline using EEG brainwave data to distinguish Wake, NREM, and REM states. Cleaned and standardized high-dimensional neural signals, generated waveform and frequency-domain visualizations, and extracted meaningful signal patterns through power spectrum analysis. Applied PCA and UMAP for dimensionality reduction, then trained Support Vector Machine models to classify sleep stages and compare cluster separability across embedding methods.
